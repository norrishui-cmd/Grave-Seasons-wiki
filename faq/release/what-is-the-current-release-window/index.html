Release FAQ

What is the current Grave Seasons release window?

Blumhouse Games currently lists Grave Seasons for Fall 2026, while Steam uses the broader year 2026.

Quick answer status

  • Answer type: confirmed current information
  • Category: Release Date
  • Last reviewed: July 19, 2026
  • Source class: official publisher page

What is confirmed

The previously announced August date is no longer current. No replacement day has been published, so Fall 2026 is the narrowest reliable public window.

Why this matters

Release information changes quickly, so this FAQ distinguishes the publisher's current window from storefront placeholders. Dates, prices, editions, preload times, and regional unlocks are updated only from Perfect Garbage, Blumhouse Games, or an official platform store.

What remains outside this answer

Unpublished values, schedules, item names, routes, rewards, campaign identities, and platform behavior are not inferred. This page will add exact steps or tables only after an official update or repeatable test in the retail build provides a concrete answer.

Source and update policy

Current information is checked against the official publisher page. Older announcements and cached storefront data are not allowed to override a newer official status. Corrections retain the same canonical URL so search engines and players reach one maintained answer.
