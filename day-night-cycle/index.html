Time System

Grave Seasons Day-Night Cycle

Official text points to a 24+ hour day and night experience, which means time is a core planning system. Farming, social routing, investigation, and safety all compete for the same calendar and stamina budget.

Time Planning Fields

  • Day segment, activity type, stamina cost, resident availability, shop timing, and travel route.
  • Night-only opportunities, suspicious movements, break-in windows, and possible victim protection checks.
  • Season and weather modifiers once exact in-game data is available.

Spoiler-Safe Use

The page should teach players when to observe, farm, socialize, and investigate without listing active killer behavior by default. Full-spoiler night routes can be gated behind clear warnings after launch.
